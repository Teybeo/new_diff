Notes sur le comportement des arguments :

  -C NB peut aussi s'écrire -CNB
  -U NB peut aussi s'écrire -UNB
  -q/--brief : pas d'effet sur le traitement des dossiers
  -s/--report-identical-files : pas d'effet sur le traitement des dossiers. N'empêche pas l'affichage d'autres informations
  -c/-C NB/--context[=N] : pas d'effet sur le traitement des dossiers. Incompatible avec -u/-e/-n/-y
  -u/-U NB/--unified[=N] : pas d'effet sur le traitement des dossiers. Incompatible avec -e/-n/-c/-y
  -y/--side-by-side : uniquement compatible avec le mode "classique" d'affichage. Incompatible avec -c/-u/-e/-n. S'affiche même pour un fichier identique
  -e/--ed : pas d'effet sur le traitement des dossiers. Incompatible avec -c/-u/-n
  -n/--rcs : pas d'effet sur le traitement des dossiers. Incompatible avec -c/-u/-e
  -w/--width NB : largeur des colonnes pour le monde "side by side". Fonctionnement indeterminé, peut afficher moins de colonnes que NB

Note sur -e :

  A l'envers (Changement le plus avancé dans le texte jusqu'à le moins)
  format : ligne affectée, nombre de ligne (1 étant implicite) c = changement / a = ajout / d = suppression
           En dessous, lignes à ajouter à f2 (sauf si suppression)
           "." Après l'extrait (Sauf dans le cas de -d)
