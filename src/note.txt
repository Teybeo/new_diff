Notes sur le comportement des arguments :

  -C NB peut aussi s'écrire -CNB
  -U NB peut aussi s'écrire -UNB

  -q/--brief : pas d'effet sur le traitement des dossiers
  -s/--report-identical-files : pas d'effet sur le traitement des dossiers
  -c/-C NB/--context[=N] : pas d'effet sur le traitement des dossiers. Incompatible avec -u/-e/-n/-y
  -u/-U NB/--unified[=N] : pas d'effet sur le traitement des dossiers. Incompatible avec -e/-n/-c/-y
  -y/--side-by-side : uniquement compatible avec le mode "classique" d'affichage. Incompatible avec -c/-u/-e/-n
  -e/--ed : pas d'effet sur le traitement des dossiers. Incompatible avec -c/-u/-n
  -n/--rcs : pas d'effet sur le traitement des dossiers. Incompatible avec -c/-u/-e
